# Video-Caption-Generator
Automated Transcription Pipeline: Integrates whisper.cpp via subprocess for high-performance, local speech-to-text inference and SRT generation.

Dynamic Motion Graphics: Implements programmatic "viral" typography using MoviePy 2.x with synchronized scaling, rotation, and fade transitions.

Robust Font Rendering: Features a multi-stage fallback system for TextClip generation to ensure compatibility across ImageMagick and PIL environments.

Signal Processing: Utilizes FFmpeg for standardized audio extraction (16kHz Mono PCM) required for optimal Whisper model accuracy.

Extensible Composition: Employs a modular architecture allowing for custom caption effects, manual SRT input, and seamless integration with NLE workflows.
